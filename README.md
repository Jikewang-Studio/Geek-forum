express# Geek-forum
一个极客论坛社区，前端采用html+css+jQuery+boostrap，服务端用Node(express)，数据库使用MongonDB，使用grunt实现流程化操作。

# 页面功能
- 论坛首页
- 文章详细页
- 个人信息页
- 文章编辑页
- 登录注册页
- 后台管理页
  - 用户管理页
  - 文章管理页

# 开发思路
前后端分离，先采用mock数据完成前端页面，再完成后端代码，最后连接测试

经过对自身以及技术方面的考虑，最终决定采用 Node.js 、 MongoDB、 Express 的开发环境
